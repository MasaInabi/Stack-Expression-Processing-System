# Stack Expression Processing System

## Description
This project is a JavaFX application for processing mathematical expressions using the stack data structure.  
It supports expression validation, conversion between different notations, and evaluation.

## Features
- Validate mathematical expressions
- Convert infix expressions to postfix
- Convert infix expressions to prefix
- Convert prefix and postfix expressions
- Evaluate expressions
- JavaFX graphical interface

## Technologies Used
- Java
- JavaFX
- CSS
- Stack Data Structure
- Eclipse IDE

## Project Files
- `src`: contains the Java source code
- `precedence.txt.txt`: contains operator precedence data
- `domain.txt.txt`: contains domain/input data

## How to Run
1. Download or clone the project.
2. Open it in Eclipse.
3. Make sure JavaFX is configured.
4. Keep the text files in the project folder.
5. Run `Main.java`.
